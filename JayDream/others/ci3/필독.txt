ci3 환경
application/libraries 밑에 JayDream 복사 777필수
js,resource 폴더는 필요없음

루트디렉토리 바로밑에 JayDream 복사 777 필수 * application 폴더와 동일선상에 두면됌
js,resource 폴더만 필요함

controller 추가  JayDreamController

routes에 통신 추가해주기
// 공통 rest API
$route['JayDream/api'] = "JayDreamController/method";

csrf or 403이 뜬다면 application/config/config.php 내
$config['csrf_exclude_uris'] = array("JayDream/api") 추가해줘야함

다른 컨트롤러에서 JayDream 불러와서 사용하고싶은경우 생성자 함수내
include_once(APPPATH . 'libraries/JayDream/require.php');
$this->app = new JayDream\App();
추가하여 전역 변수로 사용 가능
