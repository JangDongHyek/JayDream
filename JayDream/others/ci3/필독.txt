루트디렉토리 (application 폴더와 동일선상)에 JayDream 폴더 삽입

*1
apache 일경우 JayDream 폴더 예외 추가
RewriteRule ^JayDream/ - [L]

Nginx 일경우 서버블락에 해당 블락 추가
왠만하면 /etc/nginx/sites-available/default.conf 파일임
location ^~ /JayDream/ {
    try_files $uri $uri/ =404;

    location ~ \.php$ {
        # 해당부분은 php 블락을 복사해서 그대로 붙이면 됌
    }
}

*2
컨트롤러에서 사용시 밑에 구문으로 사용하면 된다
require_once FCPATH . 'JayDream/require.php';
$this->app = new JayDream\App();
